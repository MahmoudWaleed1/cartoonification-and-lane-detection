# CSE 359 - Computer Vision Projects

This repository contains coursework for **CSE 359: Computer Vision** at Alexandria University, Faculty of Engineering.
Each assignment focuses on specific aspects of computer vision, progressing from fundamental to advanced techniques.

**Instructor:** Dr. Marwan Torki

## Assignments

### Assignment 1: Cartoonification and Road Lane Detection
This assignment introduces foundational computer vision techniques through two practical tasks:
1. **Image Cartoonification** using filtering, edge detection, and stylization methods.
2. **Road Lane Detection** using image preprocessing and lane line extraction pipelines.

Project structure:
- `Assignment-1_cartoonification-and-lane-detection/Cartoonification/cartoon.ipynb`
- `Assignment-1_cartoonification-and-lane-detection/RoadLane-Detection/Road_Lane-Detection.ipynb`
